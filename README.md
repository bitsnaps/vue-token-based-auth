# Vue Token Based Authentication

## Project setup

```
npm install
```

### Compiles and hot-reloads for development & starts backend server
P.S. This will runs the node app on `http://localhost:3000`
```
npm run start
```

## Running the Vue App
This will run the vue app on `http://localhost:3001`:
```
npm run serve
```
